# liu-UI

基于Vue2的、适用于中台化开发的组件库

## 引入

 ⚠️ 持续开发中，暂未发布到NPM

## 开发

 `git clone xxxx`

 `cd liu-UI`

 `npm run dev`
 
open  `http://localhost:8088`
